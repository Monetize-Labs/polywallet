polywallet

No wallets are built the sameâ€”Polywallet resolves the ambiguities faced by developers using Bitcoin SV wallets.

Polywallet is an abstraction that enables you to keep your applications wallet agnostic and allow your customers the choice to select whichever wallet desired. Adapters for HandCash, RelayX and Invisible Money Button are currently implemented for EncryptablePaymailWallet, the interface that your applications can depend on.

Install (npm)

 
# Core (required)
npm install @monetize/polywallet-core
# Authentication modal (optional)
npm install @monetize/polywallet-modal
# Wallets (optional)
npm install @monetize/polywallet-handcash
npm install @monetize/polywallet-money-button
npm install @monetize/polywallet-relayx
npm install @monetize/polywallet-sensilet
npm install @monetize/polywallet-twetch
npm install @monetize/polywallet-volt

Install (CDN)

 
<script src="https://unpkg.com/@monetize/polywallet"></script>

Usage

Import (npm)

 
import { Polywallet, WalletBuilder } from '@monetize/polywallet-core';
import { PolywalletModal } from '@monetize/polywallet-modal';
import { HandCashWallet } from '@monetize/polywallet-handcash';
import { MoneyButtonWallet } from '@monetize/polywallet-money-button';
import { RelayXWallet } from '@monetize/polywallet-relayx';
import { SensiletWallet } from '@monetize/polywallet-sensilet';
import { TwetchWallet } from '@monetize/polywallet-twetch';
import { VoltWallet } from '@monetize/polywallet-volt';

Import (CDN)

All exported members will be exposed under the global variable Polywallet. If you are using @monetize/polywallet-modal, its will be loaded stylesheet automatically.

Wallet Builders

 
import { Polywallet, WalletBuilder } from '@monetize/polywallet-core';
import { PolywalletModal } from '@monetize/polywallet-modal';
import { HandCashWallet } from '@monetize/polywallet-handcash';
import { MoneyButtonWallet } from '@monetize/polywallet-money-button';
import { RelayXWallet } from '@monetize/polywallet-relayx';
import { SensiletWallet } from '@monetize/polywallet-sensilet';
import { TwetchWallet } from '@monetize/polywallet-twetch';
import { VoltWallet } from '@monetize/polywallet-volt';

const wallet = await new WalletBuilder<Polywallet.Wallet>()
  .with(HandCashWallet, { appId: 'APP_ID', appSecret: 'APP_SECRET' })
  .with(MoneyButtonWallet, { clientIdentifier: 'CLIENT_IDENTIFIER' })
  .with(RelayXWallet, { bitcomPrefix: 'BITCOM_PREFIX' })
  .with(SensiletWallet)
  .with(TwetchWallet)
  .with(VoltWallet)
  .withModal(PolywalletModal, {
    mode: 'COMPACT',
    theme: 'LIGHT',
  })
  .build();

const paymailWallet = await new WalletBuilder<Polywallet.PaymailWallet>()
  .with(HandCashWallet, { appId: 'APP_ID', appSecret: 'APP_SECRET' })
  .with(MoneyButtonWallet, { clientIdentifier: 'CLIENT_IDENTIFIER' })
  .with(RelayXWallet, { bitcomPrefix: 'BITCOM_PREFIX' })
  .with(SensiletWallet)
  .withModal(PolywalletModal, {
    mode: 'STANDARD',
    theme: 'DARK',
    afterClosed: () => console.log('Modal is closed!'),
  })
  .build();

const paymailWalletWithoutBalance =
  await new WalletBuilder<Polywallet.PaymailWalletWithoutBalance>()
    .with(HandCashWallet, { appId: 'APP_ID', appSecret: 'APP_SECRET' })
    .with(MoneyButtonWallet, { clientIdentifier: 'CLIENT_IDENTIFIER' })
    .with(RelayXWallet, { bitcomPrefix: 'BITCOM_PREFIX' })
    .with(SensiletWallet)
    .with(TwetchWallet)
    .withHandler((wallets) => {
      console.log('Authenticate manually!');
      wallets[0].authenticate();
    })
    .build();

const encryptablePaymailWallet =
  await new WalletBuilder<Polywallet.EncryptablePaymailWallet>()
    .with(HandCashWallet, { appId: 'APP_ID', appSecret: 'APP_SECRET' })
    .with(MoneyButtonWallet, { clientIdentifier: 'CLIENT_IDENTIFIER' })
    .with(RelayXWallet, { bitcomPrefix: 'BITCOM_PREFIX' })
    .withModal(PolywalletModal)
    .build();
// Once we have the EncryptablePaymailWallet, we can do all these things
// regardless of which wallet the customer chose.
const meta = [109, 101, 116, 97];
await encryptablePaymailWallet.sign(meta);
await encryptablePaymailWallet.decrypt(await wallet.encrypt(meta));
await encryptablePaymailWallet.sendToBitcoinAddress(
  '11A1zP1eP5QGefi2DMPTfTL5SLmv7DivfNa',
  1,
);

Modal

The polywallet modal can also be opened and closed manually.

import { PolywalletModal } from '@monetize/polywallet-modal';

const wallet = new AnyWallet();
const modal = new PolywalletModal({
  mode: 'STANDARD',
  theme: 'DARK',
  afterClosed: () => console.log('Modal is closed!'),
});

modal.open([wallet]);
modal.close();
